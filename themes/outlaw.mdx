# 法外狂徒（+1敏捷）

:::origin[[英文原文](https://www.aonsrd.com/Themes.aspx?ItemName=Outlaw)]
:::

:::starfinder
由于你过去的罪行或当前的不法行为，你在契约世界的某个地方是一个被通缉的人。你可能甚至没有罪，并正在努力洗清你的名声。或者你可能完全承认自己是个罪犯，但认为你违反的法律是不公正的。无论哪种情况，登上前往边缘世界的星际飞船可能正是你需要的，直到风头过去——或者直到你被拖进监狱。
:::

### 主题知识（1级）

你与隐秘的秘密和暗巷交易有着紧密的联系，既了解关键人物，也掌握了一些实用的技能。在回忆关于犯罪地下世界的知识时，将文化检定的DC降低5。巧手成为你的本职技能，但如果它已经是你1级时选择的职业的本职技能，则你改为在巧手检定上获得+1加值。此外，在角色创建时，你的敏捷属性获得+1调整值。

### 法律腐败（6级）

你的地下世界联系人拥有强大的影响力，能够与企业和当局打交道，并几乎可以帮你摆脱任何法律麻烦——只要你愿意支付合适的价钱。根据犯罪的严重程度，费用可能在500信用点x你的角色等级到10,000信用点x你的角色等级之间。

### 黑市联系（12级）

你有一些联系人，能够隐秘而安静地将各种商品运送到几乎任何你能想象的目的地。你可以在任何城市以通常价格出售商品，即使这些商品是非法的或对当地人来说过于奢侈而无法负担。此外，你可以以比通常价格高10%的费用购买商品，并将其运送到与一个你熟悉的城市在同一星系内的偏远交接点（例如靠近冒险地点）。交付时间至少需要城市与交接点之间的旅程时间——通常更长。

### 大师级法外之徒（18级）

组织阴影中的计划是你的专长之一，而这样做就像一剂甜美的肾上腺素。每天最多两次，在你花费至少10分钟策划一次重大盗窃、骗局或其他犯罪（这不视为休息以恢复耐力值）并成功完成至少一项实施该计划的行动后，你恢复1点决心点。
