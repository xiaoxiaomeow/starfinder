# 偶像（+1魅力）

:::origin[[英文原文](https://www.aonsrd.com/Themes.aspx?ItemName=Icon)]
:::

:::starfinder
多亏了星际传输和漂移旅行，银河系比以往任何时候都更加紧密，这种连通性也助你登上了名人的地位。你可能是一位著名的表演者，也可能是一位备受赞誉的科学家，但无论如何，你在契约世界及其相关星系中都会被人认出。你前往未知世界的理由可能是为了进一步扩大你的名声，也可能是为了逃避聚光灯的追逐。
:::

### 主题知识（1级）

选择一个专业技能。你深深融入了你所代表的标志性专业的文化中。当你尝试使用专业或文化技能回忆关于你专业中的其他标志性人物或专业文化细节的知识时，将检定的DC降低5。你选择的专业技能检定获得+1加值。文化成为你的本职技能，但如果它已经是你1级时选择的职业的本职技能，则你改为在文化检定上获得+1加值。此外，在角色创建时，你的魅力属性获得+1调整值。

### 名人（6级）

你的名声足以让几乎每个人都听说过你，或者能快速找到关于你的信息（认出你的名字需要进行DC10的文化检定，而仅凭外貌认出你需要进行DC20的文化检定）。在你所代表的标志性专业的追随者中，由于你的名人身份，你既有粉丝也有反对者。如果你在寻找一个像“能治疗这种疾病的医生”这样的普通人，你几乎总能找到一个粉丝，并且他们对你的态度初始为友好或乐于助人；这需要花费2d4小时。在GM的裁量下，粉丝可能会为你提供折扣服务（虽然不包括商品），甚至免费服务。

### 超级名人（12级）

你的声誉增长到你的名字无处不在的程度。认出你的文化检定的DC降低到5（或仅凭外貌认出你的DC降低到10），并且找到一个符合普通描述的粉丝只需1d4小时。此外，在购买商品时粉丝为你提供10%的折扣。

### 大师级偶像（18级）

每天最多两次，你可以与公众互动，谈论你的专业（通常是在表演期间，例如音乐会，但有时如果你的专业不需要观众，则可以在之后的新闻发布会上进行），总共至少10分钟，以恢复1点决心点。
