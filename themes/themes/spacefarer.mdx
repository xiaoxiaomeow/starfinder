# 星际旅者（+1体质）

:::origin[[英文原文](https://www.aonsrd.com/Themes.aspx?ItemName=Spacefarer)]
:::

:::starfinder
你对星际旅行的渴望无法被满足。你渴望踏上遥远星球的冒险，探索它的秘密。你往往以勇气和坚韧迎接每一个新机会，自信于你多样的技能将助你渡过难关。也许你只是享受与同伴一起旅行的乐趣，或者你只是为了用各种外星战利品填满你的口袋！
:::

### 主题知识（1级）

你对遥远的世界充满痴迷，总是将你所学到的关于新奇和陌生地方的一切都记在脑海中，以便在需要时回忆起来。此外，你利用对生物学和地形的了解来抵御外星危险。在回忆关于陌生新世界或太空特征的物理科学检定时，将DC降低5。物理科学成为你的本职技能，但如果它已经是你1级时选择的职业的本职技能，则你改为在物理科学检定上获得+1加值。此外，在角色创建时，你的体质属性获得+1调整值。

### 热心的涉猎者（6级）

在你的旅途中，你学到了许多关于各种事物的技巧，即使你并未正式学习过它们，你常常可以利用这种逻辑和直觉为自己带来优势。如果你在某个技能上没有技能等级，你在该技能检定上获得+2加值。此能力不允许你尝试仅限受训技能（见第134页）的检定。

### 万事通（12级）

只要你用心去做，你几乎可以完成任何事情，而且你从不让缺乏正式指导成为你处理任务的障碍。你可以使用所有技能，即使你通常无法这样做，并且当你在没有技能等级的技能检定上掷出天然20时，你从热心的涉猎者中获得的加值增加到+4。

### 大师级探索者（18级）

科学地记录一个新地方的每一个微小细节——从明显的颜色和坡度到气压、地震和其他精细读数——对你来说绝对令人振奋。每天最多两次，在一个未被探索的星球上（通常是一个与契约世界没有接触的星球，尽管它不一定是你自己发现的），你可以花费10分钟探索、绘制地图并记录一个新的地理特征，以恢复1点决心点；这不视为休息以恢复耐力点。
