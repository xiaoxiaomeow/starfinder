# 王牌飞行员（+1敏捷）

:::origin[[英文原文](https://www.aonsrd.com/Themes.aspx?ItemName=Ace%20Pilot)]
:::

:::starfinder
你最舒适的地方是在载具的操控台上，无论是驾驶星舰穿越漆黑的太空，还是驾驶地面车辆在树木间、巨石旁和尘土飞扬的荒原上疾驰。你可能是一支精英军事部队的成员，接受过严格的训练课程。或者，你可能是一个完全业余的爱好者，但拥有与生俱来的天赋，使你成为备受钦佩的高手。
:::

### 主题知识（1级）

你对星舰和载具充满热情，几乎将所有相关的知识都牢记于心。在进行文化检定时，回忆关于星舰和载具模型、零件以及著名飞行员的知识的DC降低5。驾驶成为你的本职技能，但如果它已经是你1级职业的本职技能，则改为在驾驶检定上获得+1加值。此外，在角色创建时，你的敏捷属性获得+1调整值。

### 独狼（6级）

你至少对星舰上的每个角色都有一定的了解，并且可以在紧急情况下替代某些任务。每当你在星舰战斗期间或直接修理或维护星舰时需要尝试技能检定时，你可以将驾驶技能的一半等级视为该检定的技能等级（如果这样更有利的话）。由于你如同拥有相关技能的等级，因此在此检定中你被视为受训。

### 需要速度！（12级）

驾驶载具高速行驶让你感到兴奋，你可以轻松应对高速操作，而其他飞行员可能会失控。你在驾驶载具时进行的驾驶检定的任何减值减少1。当你在载具追逐中采取双倍机动动作时（见第283页），每个动作的减值减少1。对于驾驶检定因失败5点或更多而受到惩罚的场合，你只有在失败10点或更多时才会受到惩罚。

### 大师级飞行员（18级）

你的飞行成就激励着你，赋予你新的目标和热情。每天最多两次，当你在星舰战斗中作为飞行员击败一个重要敌人或在载具追逐中成功（意味着你逃脱了追捕者，或捕获或击败了对手）时，你恢复1点决心点。
