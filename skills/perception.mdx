# 察觉（感）

:::origin[[英文原文](https://www.aonsrd.com/Skills.aspx?ItemName=Perception)]
:::

你可以运用所有感官（听觉、味觉、触觉、视觉和嗅觉）来察觉危险、辨别细节或搜寻隐藏的物体与生物。

## 注意

你可以使用察觉技能来注意周围发生的事件。这是察觉技能最基本的用途，具体应用由GM决定。你可能需要进行察觉检定以判断能否在突袭轮行动、用余光发现重要事物，或是意识到附近有隐藏生物（但你无法察觉隐形生物，除非它揭晓了它存在，详见第264页）。用于注意的察觉检定通常不需要动作，但你必须保持清醒且至少部分感官可用。检定DC由GM设定，且此类检定不可取20。

注意任务的察觉检定DC由GM根据情境决定，也可能根据其它状态调整，[搜索](#搜索)任务的表格中提供了一些示例情景的典型DC。

## 识破伪装

作为移动动作，你可以通过察觉检定识破生物的伪装。若你特别警觉区域内可能的伪装者，GM可能允许你无需动作进行检定。该检定与伪装者的[易容](/skills/disguise)检定对抗。若成功，你会意识到该生物经过伪装，但不一定知晓其真实身份。若伪装者模仿的是你认识的特定个体，GM可能给予察觉检定加值。

能看穿幻术和全息影像的预言魔法或扫描仪无法识破普通伪装，但它们能让伪装中的幻术或全息影像提供的加值失效。

识破伪装任务的察觉检定DC会根据观察者对伪装对象熟悉程度调整，以及由DM决定的其它环境调整。

:::table
| 熟悉程度 | DC调整值 |
| :---- | :----: |
| 亲密关系 | -10 |
| 密友 | -8 |
| 朋友/熟人 | -6 |
| 面熟 | -4 |
:::

## 搜索

作为移动动作，你可以使用察觉技能来搜寻特定事物，例如寻找已揭晓了它存在的隐形生物、你知道藏身该区域的藏匿生物，或是探查附近的陷阱或危机。你也可以通过搜索来发现区域内可能被隐藏或不易察觉的有趣事物，比如隐蔽的货舱或藏匿的宝藏。这种情况下，搜索不超过20尺x20尺的区域需要1分钟（若GM判定该区域特别杂乱或复杂，范围可能更小）。检定DC由生物的隐匿检定、相关陷阱或危机的数值、或GM决定。若非战斗状态，你可以对察觉检定取20来搜索。

搜索任务的察觉检定DC可能由GM根据环境和其他状态进行调整。下表提供了一些常见情景及其典型DC值。

:::table
| 情景 | DC |
| :---- | :----: |
| 听到战斗声响 | 0 |
| 察觉强烈气味 | 0 |
| 听清对话内容 | 0 |
| 发现视野内的生物 | 0 |
| 判断食物是否变质 | 5 |
| 听到小型或中型生物脚步声 | 10 |
| 听清耳语对话内容 | 15 |
| 找到普通无标记货舱门 | 15 |
| 听到自动门开启声 | 20 |
| 发现普通暗格 | 20 |
| 察觉手枪出鞘声 | 20 |
| 感知地下5尺处的掘地生物 | 25 |
| 发现扒窃行为 | 与[巧手](/skills/sleight%20of%20hand)检定对抗 |
| 察觉隐匿中的生物 | 与[隐匿](/skills/stealth)检定对抗 |
| 发现隐藏陷阱 | 依陷阱而定 |
:::
