# 唬骗（魅）

:::origin[[英文原文](https://www.aonsrd.com/Skills.aspx?ItemName=Bluff)]
:::

你可以通过言语和行动分神、误导对手、编造可信的谎言以及传递秘密信息。

## 分神

作为一个移动动作，你可以使用唬骗技能分神。你的唬骗检定将与目标生物的[察言观色](/skills/sense%20motive)检定对抗。若成功，你可以选择如同拥有掩蔽或隐蔽般尝试隐匿技能的[躲藏](/skills/stealth#躲藏)任务，或在巧手技能的[掌中藏物](/skills/sleight%20of%20hand#掌中藏物)任务中获得+10加值。若身处拥挤空间站长廊等场合，GM可能要求与多个生物的[察言观色](/skills/sense%20motive)检定对抗——此时仅对检定结果低于你唬骗结果的生物生效。

## 虚招

作为标准动作，你可以在战斗中使用唬骗技能进行虚招，使目标在你下回合结束前，针对你的下一次攻击被视为措手不及。该检定DC为10+目标[察言观色](/skills/sense%20motive)技能总加值，或15+1.5x目标CR中的较高者。无法对没有智力属性的生物使用虚招，且该唬骗检定不可取10或取20。

## 说谎

你可以通过唬骗技能欺骗他人或编造可信谎言。战斗中快速简单的谎言属于战斗对话范畴；其他情况下说谎至少需整轮动作，复杂谎言耗时由GM裁定。若目标持怀疑态度或专注识破谎言（如同[察言观色](/skills/sense%20motive)技能的识破欺骗任务），需与其[察言观色](/skills/sense%20motive)检定对抗；否则DC为10+目标[察言观色](/skills/sense%20motive)技能总加值。成功后目标会相信你在说实话，至少直到出现反证。GM可判定某些荒谬谎言绝无可能让人相信。

唬骗检定的DC会根据目标初始态度（见[交涉](/skills/diplomacy)）及GM判定的其他情境（如谎言合理性）调整。

| 初始态度 | DC调整值 |
| :---- | :----: |
| 敌对 | +10 |
| 不友善 | +5 |
| 漠不关心 | +0 |
| 友善 | -5 |
| 乐于帮助 | -5 |

## 传递密语

你可以使用唬骗技能向盟友传递密语，让其它人不知道密语的真正含义。在战斗中这样做是战斗对话的一部分。这个检定的DC对于简单信息为15，复杂信息为20（由GM判定）。偷听者可进行[察言观色](/skills/sense%20motive)对抗检定以获悉大意。该唬骗检定不可取20。
