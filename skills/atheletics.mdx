# 运动（力；减）

:::origin[[英文原文](https://www.aonsrd.com/Skills.aspx?ItemName=Athletics)]
:::

你可以攀爬垂直表面、跃过障碍以及游泳。

## 攀爬

作为移动动作的一部分，你可以使用运动技能爬上、爬下或者爬过斜坡、墙壁或其他陡峭斜面。只要存在着力点，你甚至可以在天花板上攀爬，但无法攀爬完全光滑的表面。若检定成功，你以一半陆地速度移动。若检定失败但差值不超过4，你未能前进。若差值达到5或更高，你会坠落。

攀爬的运动检定不可取20。攀爬至少需要双手，但你可以用单手攀附墙壁，同时施法、使用小型火器或执行其他仅需单手的动作。攀爬或攀附墙壁时，你处于措手不及状态。若拥有攀爬速度（见第259页），你在攀爬运动检定上获得+8加值，且除非处于危险环境，否则无需进行攀爬检定。

攀爬运动检定的DC基于攀爬对象，但也会因重力、风力、表面状况等环境因素调整。这些调整值可累计叠加，使用所有适用的调整值。

:::table
| 攀爬对象 | DC |
| :---- | :----: |
| 45度以上斜坡、梯子或有墙壁支撑的结绳 | 5 |
| 有墙壁支撑的无结绳索或缆绳<br/>无墙壁支撑的结绳 | 10 |
| 粗糙表面或有充足着力点的墙壁，如天然岩壁、洞穴壁或带缆绳的人工墙<br/>无墙壁支撑的无结绳索或缆绳<br/>用手悬挂的岩架 | 15 |
| 着力点狭窄的不规则表面，如石墙、砖墙 | 20 |
| 偶有着力点的相对光滑表面。如空间站墙壁、冰墙 | 25 |
| 仅有着力点的悬垂物或天花板 | 30 |
| 完全光滑的表面 | - |
:::

:::table
| 环境调整 | DC调整值 |
| :---- | :----: |
| 以两个相对的墙为支撑 | -10 |
| 在角落攀爬或以直角墙为支撑 | -5 |
| 零重力/低重力环境 | -5 |
| 高重力环境* | +5 |
| 潮湿/轻微湿滑的墙面 | +2 |
| 结冰/严重湿滑的墙面* | +5 |
| 强风环境（21-30英里/时） | +2 |
| 烈风环境（31-50英里/时）* | +5 |
| 风暴环境（51-74英里/时）* | +10 |
| 飓风环境（75-174英里/时）* | +20 |
| 龙卷风环境（175+英里/时）* | +30 |

*危险环境：拥有攀爬速度的生物在此类条件下仍需检定。
:::

## 跳跃

作为移动动作的一部分，你可以使用运动技能进行水平或垂直跳跃，距离不得超过剩余移动力。若起跳前有10尺助跑，检定DC等于水平跳跃尺数，或垂直跳跃尺数×4。

若无助跑，检定DC翻倍。环境调整值与特技技能平衡检定相同（见第135页）。若检定失败，你会坠落。若差值达到5或更高，即使未受坠落伤害也会倒地。陆地速度35尺以上的生物在跳跃检定时获得+4加值，速度每超过40尺10点，该加值再+4。跳跃运动检定不可取20。

## 游泳

作为移动动作的一部分，你可以使用运动技能游泳。检定成功时，你在水或类似流体中以一半陆地速度移动。若检定失败但差值不超过4，你未能前进。若差值达到5或更高，你会沉入水中（需闭气否则溺水，见第404页）。若无游泳速度（见第259页），每小时需通过DC20运动检定，否则因疲劳受到1d6非致命伤害。拥有游泳速度的生物在所有游泳检定上获得+8加值，且除非处于危险环境，否则无需进行游泳检定。

游泳运动检定的DC基于环境条件，但也会因水流、水中的残骸等因素调整。这些调整值可累计叠加，使用所有适用的调整值。

:::table
| 水域条件 | DC |
| :---- | :----: |
| 平静水域 | 10 |
| 湍急水域 | 15 |
| 暴风水域* | 20 |
| 漩涡水域* | 30 |

*除非拥有游泳速度，否则在暴风/漩涡水域中即使未受干扰也不能取10。
:::

:::table
| 环境调整 | DC调整值 |
| :---- | :----: |
| 轻微残骸/扰动 | +2 |
| 重度残骸/扰动* | +5 |
| 顺流游泳* | +5 |
| 逆流游泳* | +10 |
| 对抗喷射水流/强逆流* | +15 |

*危险环境：拥有游泳速度的生物在此类条件下仍需检定。
:::
