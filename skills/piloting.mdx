# 驾驶（敏）

你懂得如何驾驶载具、操控星舰以及导航。

## 发射星舰炮

在星舰战斗中尝试炮术检定时（见第320页），你可以使用驾驶技能等级或基础攻击加值来计算攻击掷骰。

## 导航

你可以使用驾驶技能进行导航或星际航行。这让你能引导载具或星舰朝向目标航向，并规划更长程的航线。为到达你经常造访的恒星系规划航线通常需要通过DC10的驾驶检定并花费10分钟。为较不熟悉的恒星系规划航线更为困难，且需要关于目标星系的信息；若你当前迷失方向，导航也会更加困难。

若你在规划恒星系间航线的检定中失败但差值不超过9，你会意识到航线规划有误，必须重新尝试检定才能启程。若检定失败差值达到10或更多，你将不会察觉计算错误，抵达目的地的时间会比正常情况更长（漂移旅行通常额外增加1d6天）。根据GM的判断，你也可能抵达陌生的恒星系（从该处规划前往实际目的地的航线通常需要通过DC25的驾驶检定），或是当抵达预定目的地时，你的星舰引擎可能获得故障严重损伤状态（见第321页）。

你对太空区域或行星的熟悉程度决定了导航或星际航行的驾驶检定DC。GM可根据起始地点与目的地的信息量，以及该地点是否特别难以导航（例如无路可循的沙漠或奇异星云）来调整这些DC（通常增减5到10）。

:::table
| 熟悉程度 | DC |
| :---- | :----: |
| 频繁造访或家园基地 | 10 |
| 偶尔造访 | 15 |
| 陌生区域 | 25 |
:::

## 驾驶星舰

若你在星舰战斗中担任驾驶员角色，需使用驾驶技能进行机动、尝试特技动作及其他星舰操控。完整规则见第324页。

## 驾驶载具

驾驶载具时（见第278页），你需要尝试驾驶检定以全速安全竞逐、完成机动动作、参与载具战斗以及在追逐中进行机动动作。更先进的载具更难驾驶，因此大多数载具驾驶检定的DC会随载具的物品等级提升。
