# 易容（魅）

:::origin[[英文原文](https://www.aonsrd.com/Skills.aspx?ItemName=Disguise)]
:::

你能改变外貌以融入环境并欺骗他人，无论是为了渗透还是其他目的。

## 改变外貌

你可以使用易容技能，花费1d3×10分钟，配合易容工具、通过施放易容术等法术、或使用全息皮肤等科技装置改变外貌。GM会秘密进行易容检定，因此你无法确知伪装效果。该检定将与可能识破你伪装者的[察觉](/skills/perception)检定对抗。若你未引起他人注意，其他生物通常不会主动进行察觉检定来识破伪装。若生物对可疑行为特别警觉（例如星舰或空间站上的安保人员），则假定这些观察者在其察觉检定上取10。

伪装效果取决于你改变外貌的程度。若无变形魔法或技术协助，你仅能伪装成与自身体型相同或相差一个体型的生物。这不会改变你的实际体型或触及范围。伪装具有笼统性——你无法伪装成特定个体。

某些法术（如易容术）可为易容检定提供+10加值。

易容检定的DC会根据伪装类型及GM判定的其他情况进行调整。下列调整值可累计叠加，使用所有适用的调整值。

:::table
| 伪装 | DC调整值 |
| :---- | :----: |
| 仅改变细微细节 | -5 |
| 改变主要特征 | +2至+5 |
| 伪装为同生物类型的其他种族 | +2至+8 |
| 伪装为其他生物类型 | +10 |
| 伪装为其他体型 | +10 |
:::
