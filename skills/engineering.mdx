# 工程（智；训）

你可以辨识、建造、修理或解除科技装置；评估结构和机械的稳定性；以及正确安装和拆除爆炸物。若你在尝试工程检定时没有工程工具包，检定将承受-2减值。

## 安装爆炸物

你可以使用工程技能通过引爆器安装爆炸物（见218页）。连接引爆器并设置爆炸物需花费1分钟。该检定DC通常为10。若检定失败，你可以重新尝试安装。若检定失败且差值达到5或更高，爆炸物将被提前触发。

你也可以尝试建造更难拆除的爆炸物。为此，选择一个目标拆除DC（DC应以5为增量，最低DC为15）。该DC将成为你安装爆炸物的目标DC，同时也是拆除爆炸物的DC（见下文解除装置）。

## 评估稳定性

你可以使用工程技能评估结构或机械装置，以确定其稳定性、可用性和结构弱点。这需要1分钟或更长时间，DC由GM决定。

使用以下基础DC进行工程检定以评估稳定性。这些DC可根据其他情况调整，例如结构的复杂度和损坏程度，由GM决定。

:::table
| 任务 | DC |
| :---- | :----: |
| 评估稳定性 | 15 |
| 确定结构弱点 | 20 |
:::

:::table
| 情况 | DC调整值 |
| :---- | :----: |
| 简单结构（绳桥或不稳定天花板） | -5 |
| 复杂结构（悬索桥或空间站墙壁） | +5 |
| 明显损坏 | -5 |
| 轻微但关键的损坏 | +5 |
| 蓄意破坏 | +10 |
:::

## 制造科技物品

若你在工程技能上有足够等级，可以制造科技装置或物品（包括计算机）。造物规则见235页。

## 解除装置

你可以使用工程技能解除锁、陷阱、机械或科技装置，或拆除爆炸物，只要装置无人看管且你可以接触到它。所需时间取决于装置的复杂度，通常至少需要一个整轮动作。检定DC由GM根据装置复杂度决定。对于极其复杂的装置或系统，GM可能要求多次检定。GM秘密掷骰解除装置的工程检定，因此你不一定知道尝试是否成功。若成功，装置被解除。若失败且发现错误，可以重新尝试。若失败且差值达到5或更高，将出现意外。若装置是爆炸物或陷阱，它将触发。若尝试某种破坏，你会认为装置已解除，但它仍能正常工作。

你也可以使用解除装置任务来使装置暂时正常工作，之后才被解除。这将使检定DC增加5。若想不留篡改痕迹，DC再增加5。若检定成功，你可以使装置在至多你工程技能每等级1轮后失效。若检定失败且差值达到5或更高，结果等同于你仅尝试普通地解除装置。

由于危险性，你不能在解除装置的工程检定上取20。

解除装置的工程检定DC基于装置复杂度。下表提供基础DC、示例装置及解除时间。GM可根据其他情况调整这些DC和时间。具有备份或类似安全措施的系统DC可比表中高1至5。

:::table
| 装置 | 示例 | 时间 | DC |
| :---- | :---- | :----: | :----: |
| 简单装置 | 卡住门 | 1轮 | 10 |
| 棘手装置 | 破坏简单推进系统 | 1d4轮 | 15 |
| 困难装置 | 拆除或重置哨戒炮或类似陷阱 | 2d4轮 | 20 |
| 复杂装置 | 从控制面板拆除爆炸物或安全系统 | 2d4轮 | 25 |
| 装备 | 关闭护甲升级、动力装甲或武器 | 2d4轮 | 15+1.5x物品等级 |
| 简单锁 | - | 1轮 | 20 |
| 普通锁 | - | 1轮 | 25 |
| 精良锁 | - | 1轮 | 30 |
| 超级锁 | - | 1轮 | 40 |
:::

## 辨识生物

你可以使用工程技能辨识具有科技亚种的构装体，例如机器人（见[辨识生物](/core_rulebook/chapter%205#辨识生物)）。

## 辨识科技

你可以使用工程技能辨识科技物品和装置的属性和用途，例如星舰和武器，以及外星科技。通常，辨识契约世界中常见的相对简单的科技物品（例如[第7章](/core_rulebook/chapter%207)中的物品）无需检定。你可以在辨识科技的工程检定上取20，但必须有研究手段，例如接入信息网络或下载数据集。

辨识科技的工程检定DC基于物品稀有度。

:::table
| 物品稀有度 | DC |
| :---- | :----: |
| 常见复杂科技（契约世界星舰或物品） | 5+1.5x物品等级 |
| 较不常见科技（非契约世界星舰或物品） | 10+1.5x物品等级 |
| 稀有、古老或外星科技 | 15+1.5x物品等级 |
:::

## 修理物品

你可以使用工程技能修理机械、科技或混合物品或装备，只要你能接触它。所需时间通常取决于物品复杂度。如果物品或装备是你制造的，修理时间减半。检定DC由GM根据物品复杂度决定。若成功，你恢复等同于工程检定结果的生命值。若失败且差值达到10或更高，物品进一步损坏，受到1d4伤害；此伤害不会使物品生命值低于1。

若物品或装备受伤但未破损，可免费修理。若破损但未摧毁，每次尝试修理需花费10单位通用修补剂（见233页；简单物品等级视为1）。摧毁的物品或装备无法用工程技能修理。

你不能在修理物品的工程检定上取20。

修理物品的工程检定DC基于物品复杂度。下表提供基础DC和示例物品。GM可根据其他情况调整这些DC和时间。

| 物品 | 示例 | 时间 | DC |
| :---- | :---- | :----: | :----: |
| 简单 | 门或墙 | 10分钟 | 15 |
| 复杂 | 计算机控制台 | 30分钟 | 20 |
| 装备 | 武器或护甲 | 1小时 | 15+1.5x物品等级 |
