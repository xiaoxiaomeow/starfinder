# 文化（智；训）

:::origin[[英文原文](https://www.aonsrd.com/Skills.aspx?ItemName=Culture)]
:::

你研习过银河系中大量已知文明，对文化潜流与语言有着深刻而丰富的理解。每次在文化技能上投入等级时，你学会说读写一门新语言。常见语言列表参见[常见语言](/core_rulebook/chapter%203#常见语言)。

## 破译文字

你可以使用文化技能来破译陌生语言的文字，或残缺或古体形式书写的信息。破译约250字以内的文字至少需要1分钟。GM通常会秘密进行文化检定来破译文字。若检定成功，你能理解文本的大意；若失败则无法理解。若检定失败且差值达到10或更多，你会彻底误解文本含义。除非你受训于[计算机](/skills/computer)技能且能接入信息网络或下载数据集，否则破译文字时不能取20。此时文字信息将不存在误读可能。

文化检定破译文字的DC基于文本复杂度及GM判定的其他情况：

| 复杂度 | DC |
| :---- | :---- |
| 简单信息 | 20 |
| 标准文本 | 25 |
| 复杂、异域或远古文字 | 30 |

## 回忆知识

你可以使用文化技能回忆关于某文化的习俗、法律、政府、领袖、知名居民、传说、宗教、历史及相关话题的知识（见[回忆知识](/core_rulebook/chapter%205#回忆知识)）。
