# 特技（敏；减）

你能在狭窄或危险的地形上保持平衡、挣脱束缚，并通过翻滚躲避攻击。特技技能也用于判定飞行时高难度机动动作的成功与否。

## 平衡

作为移动动作的一部分，你可以使用特技技能在不跌倒的情况下通过狭窄地面或不平地面。成功的检定允许你以一半陆地速度通过此类地面。平衡时你处于措手不及状态。若检定失败，你的移动动作会在需要开始平衡的位置前终止。若已在平衡时失败（先前回合成功过），你会倒地，且GM可判定你开始坠落，具体取决于所处地面类型。

若你正在平衡时受到伤害，必须立即以初始DC进行特技检定。成功则保持平衡（如果现在是你的回合，你可以继续移动），失败则倒地并可能坠落。平衡检定不可取20。

平衡检定DC基于地面宽度，并受环境因素调整（如坡度、地面状况）。这些调整值是叠加的，使用所有适用的调整值。

:::table
| 地面宽度 | DC |
| :---- | :----: |
| 大于3尺* | 0 |
| 3-1尺* | 5 |
| 11-7英寸 | 10 |
| 6-2英寸 | 15 |
| 小于2英寸 | 20 |

*若无DC调整使DC至少为10，则无需检定
:::

:::table
| 环境状况* | DC调整值 |
|-------------------------|---------|
| 轻微阻碍（砂砾、沙地） | +2 |
| 严重阻碍（洞穴、碎石） | +5 |
| 轻微湿滑（潮湿） | +2 |
| 严重湿滑（结冰） | +5 |
| 轻微倾斜（\<45度） | +2 |
| 严重倾斜（\>45度） | +5 |
| 轻微不稳（颠簸太空飞行）| +2 |
| 中度不稳（摇晃飞船） | +5 |
| 严重不稳（地震） | +10 |

*适用于特技技能的平衡/翻滚任务与运动技能的跳跃任务
:::

## 挣脱

你可以用特技脱离擒抱、压制或束缚。挣脱擒抱或压制是标准动作，成功则解除相应状态，DC=10+擒抱者的动能护甲等级。挣脱束缚需1分钟以上，DC基于束缚类型，有时取决于进行束缚的生物的CR（见下表）。挣脱大部分束缚可取20，但擒抱不可。

| 状况 | DC |
| :---- | :----: |
| 被擒抱或压制 | 10+擒抱者KAC |
| 被绳索捆绑 | 20+1.5×对手CR |
| 被镣铐束缚 | 30 |

## 飞行

基础飞行规则见259页。通常仅在尝试高难度机动或遭遇强风时才需特技检定（作为移动动作的一部分）。飞行机动性影响检定：笨拙机动性-8，普通机动性无调整，完美机动性+8。

以下飞行情况需特技检定且会产生失败后果。这些检定的DC也取决于当前的风和空气状态，和由DM决定的其它因素。你不能在用于飞行的特技检定上取20。

### 避免坠落伤害

如果你正在坠落但具有飞行能力，你可以用反应动作进行特技检定抵消坠落伤害。如果失败，则你正常坠落。如果你有完美机动性，你可以自动避免坠落伤害而无需检定。

### 在强风中飞行

风力越强，保持飞行就越难。在有风暴或更高（风速≥51英里/小时，见400页）的状态下，每次移动都需要检定来保持飞行。如果失败，则你无法移动。如果你失败超过5，则被按照风向（或DM决定的其它方向）吹离2d6x10尺并受到2d6钝击伤害。你不能在用于在强风中飞行的特技检定上取10。

### 悬停

通常需动量才能维持飞行。如果你想在飞行过程中原地不动或悬停，你需要用移动动作做特技检定。如果失败，则你坠落。如果你有笨拙机动性，你不能进行悬停。如果你有完美机动性，你可以自动悬停而无需检定，但你如果你通过检定则可以以迅捷动作悬停而不是移动动作（失败没有惩罚）。

飞行检定DC基于风速：

| 风力状况 | DC | 需检定？ | 可取10？ |
|-----------------------|----|----------|----------|
| 轻至中度（0-20mph） | 15 | 否 | 是 |
| 强风（21-30mph） | 17 | 否 | 是 |
| 剧烈（31-50mph） | 19 | 否 | 是 |
| 风暴（51-74mph） | 27 | 是 | 否 |
| 飓风（75-174mph） | 33 | 是 | 否 |
| 龙卷风（175+mph） | 39 | 是 | 否 |

乱流或空中有碎片时DC+5，两者兼具+10。

## 翻滚
未处于负重或超载状态（见275-276页）时，可用特技通过敌人威胁区域而不引发借机攻击。翻滚是移动动作，以半速移动。

通过威胁区域DC=15+1.5×对手CR；多个敌人时以最高CR为基准，每多一名+2。直接穿过敌人空间DC=20+1.5×对手CR，失败则停在敌人邻接格并引发借机攻击。

同轮内通过多个威胁区域时，每个区域需独立检定且后续DC+2。全速翻滚检定-10。俯身翻滚时只能移动5尺（整轮动作）且检定-5。

翻滚基准DC：

| 情况 | DC* |
|---------------------------|--------------------|
| 通过威胁区域 | 15+1.5×对手CR |
| 穿过敌人空间 | 20+1.5×对手CR |

*同轮内每多通过一个区域DC+2

环境调整值与平衡任务相同（见135页）。检定失败则正常引发借机攻击。
