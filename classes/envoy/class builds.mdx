# 职业构筑

:::origin[[英文原文](https://www.aonsrd.com/ClassBuilds.aspx?Class=Envoy)]
:::

:::starfinder[大使]
你在外交事务中代表某个特定政府，甚至偶尔会与新的生命形式接触。

**主题** [异形探索者](/themes/xenoseeker)  
**属性值** 魅力为你提供更多决心点并强化你的社交技能，而力量则让你在使用近战武器时造成更大伤害。  
**外交官即兴** [挫志嘲讽](envoy%20improvisations#挫志嘲讽ex)（1级）、[通用表达](envoy%20improvisations#通用表达ex)（1级）、[快速挫志嘲讽](envoy%20improvisations#快速挫志嘲讽ex)（4级）、[吸引火力](envoy%20improvisations#吸引火力ex)（6级）  
**专家才能** [文化通](expertise%20talents#文化通ex文化) 、[震慑气场](expertise%20talents#震慑气场ex威吓)  
**专长** 武器专攻（进阶近战武器）、武器擅长（进阶近战武器）、武器专精（进阶近战武器）  
**技能** 唬骗、威吓、文化、察言观色、交涉
:::

:::starfinder[军官]
你擅长指挥部队，无论是在舰桥上还是战场上，都能让你的部队发挥最大作用。

**主题** [王牌飞行员](/themes/ace%20pilot)  
**属性值** 敏捷是你最重要的能力值，因为你需要它来进行远程攻击命中，而魅力则为你提供更多决心点。  
**外交官即兴** [干掉他们](envoy%20improvisations#干掉他们ex)（1级）、[保持警惕](envoy%20improvisations#保持警惕ex)（1级）、[快速挫志嘲讽](envoy%20improvisations#快速挫志嘲讽ex)（4级）、[强化干掉他们](envoy%20improvisations#强化干掉他们ex)（6级）
**专家才能** [敏锐观察者](expertise%20talents#敏锐观察者ex察言观色)、[震慑气场](expertise%20talents#震慑气场ex威吓)  
**专长** 远程武器擅长、武器专攻（远程武器）、武器专精（远程武器）  
**技能** 特技、驾驶、运动、察言观色、威吓
:::

:::starfinder[谈判家]
你擅长商业，知道如何在任何情况下达成最佳交易。

**主题** [偶像](/themes/icon)  
**属性值** 魅力为你提供更多决心点并强化你的社交技能，而智力则让你更了解与你打交道的人。  
**外交官即兴** [别打脸](envoy%20improvisations#别打脸ex)（1级）、[加速](envoy%20improvisations#加速ex)（4级）、[巧思即兴](envoy%20improvisations#巧思即兴ex)（6级）、[强化加速](envoy%20improvisations#强化加速ex)（8级）  
**专家才能** [文化通](expertise%20talents#文化通ex文化) 、[圆滑顾客](expertise%20talents#圆滑顾客ex交涉)  
**专长** 精通先攻、技能专攻（交涉）、技能专攻（文化）  
**技能** 唬骗、专业（仲裁者）、文化、察言观色、交涉
:::

:::starfinder[恶棍]
虽然你是个通常只为自己着想的罪犯，但在紧要关头，你可以指望帮助你的朋友。

**主题** [法外狂徒](/themes/outlaw)  
**属性值** 魅力为你提供更多决心点并强化你的社交技能，而敏捷则帮助你生存。  
**外交官即兴** [巧妙虚招](envoy%20improvisations#巧妙虚招ex)（1级）、[小心脚下](envoy%20improvisations#小心脚下ex)（1级）、[巧妙攻击](envoy%20improvisations#巧妙攻击ex)（4级）、[小心](envoy%20improvisations#小心ex)（4级）  
**专家才能** [说服性谎言](expertise%20talents#说服性谎言ex唬骗)、[狡诈伪装](expertise%20talents#狡诈伪装ex易容)  
**专长** 精通虚招、技能专攻（唬骗）、高等虚招  
**技能** 唬骗、巧手、文化、隐匿、易容
:::
