# 欢迎来到 Starfinder

探索外星遗迹，寻找失落的魔法。在空间站阴暗的街道上，与企业术法师及其机器人军团斗智斗勇。驾驶星舰扫射由巨龙操控的攻击巡洋舰，保卫新殖民地免受致命异星掠食者的侵害，或与未知的外星帝国进行首次接触。

无论你的武器是激光步枪、动力装甲、神秘魔法，还是纯粹的口才，Starfinder 角色扮演游戏讲述了英雄们如何凭借一次次冒险，改变整个银河的命运。
