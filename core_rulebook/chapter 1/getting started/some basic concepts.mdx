# 一些基础概念

:::origin[[英文原文](https://www.aonsrd.com/Rules.aspx?ID=8)]
:::

为了更好地使用本书，你需要熟悉一些关键术语和缩写。它们贯穿全书，并且许多都是桌面角色扮演游戏中常见的术语。更详细的术语表请参阅第 512 页。

### 1d6, 1d20等
这些数字是掷骰子的缩写，表示你在确定某个变量数值时要掷哪些骰子，例如武器造成的伤害。第一个数字告诉你要掷几个骰子，而第二个数字告诉你骰子应有多少个面；如果没有第一个数字，就只掷一个骰子。例如，“掷2d8”意味着你要掷两个八面骰，而“掷d20”意味着你要掷一个二十面骰。有时，你可能需要掷d3；如果没有三面骰，你可以用d6代替——将掷出的1或2当作1，掷出的3或4当作2，掷出的5或6当作3。

### 缩写（Abbreviations）
星际探险有许多通常以缩写表示的术语，包括HP（生命值）、SP（耐力值）和RP（决心点）。如果你忘记了缩写的意思，可以在第512页开始的词汇表中查找。

### 护甲等级（Armor Class，AC）
这是一个数字，表示敌人打击你角色的难度。一个角色有两个护甲等级：能量护甲等级（EAC）和动能护甲等级（KAC）。

### 攻击掷骰（Attack Roll）
攻击掷骰是一个d20的掷骰，表示你角色尝试在战斗中攻击其他生物的行为。

### 检定（Check）
检定是一个d20的掷骰，可能会受到你角色的属性或其他数值的修正。最常见的类型是技能检定和属性检定（决定你是否成功执行某个任务），以及先攻检定（决定你在战斗中行动的顺序）。

:::info[译注]
Starfinder 中，攻击掷骰（Attack Roll）和豁免掷骰（Saving Throw）不是检定（Check）。
:::

### 战斗（Combat）
通常，战斗指的是个体角色之间的战术战斗，这种战斗在方格战斗地图上进行，详细信息可以在第8章找到。战斗也可以指星舰战斗，它使用六角地图；关于这个系统的细节，可以在第9章找到。

### 生物（Creature）
生物是故事或世界中的一个积极参与者，包括玩家角色（PC）、非玩家角色（NPC）和怪物。

### 难度等级（Difficulty Class，DC）
这是一个目标数字，生物在进行检定时必须达到或超过该数字，才能完成给定的任务。

### 遭遇（Encounter）
遭遇是给角色带来挑战的情境。这可能是一个需要获取信息的角色扮演挑战，一场物理战斗，一个陷阱或谜题，或者任何其他需要玩家动脑筋或利用角色数值的情况。角色通常因完成遭遇而获得经验点。

### 经验点（Experience Points，XP）
通常简称“经验”，它是一种记录角色通过克服挑战获得的专业知识的方式。当角色获得足够的经验点时，他们会提升等级，或“升级”（详见第26页的升级规则）。

### 游戏主持人（Game Master，GM）
游戏主持人是负责裁定规则并控制星际探险故事和世界中各种元素的玩家。GM的责任是提供一个公平且有趣的游戏——她希望其他玩家最终能成功达成目标，但只有经过英雄般的奋斗和危险之后。

### 生命值（Hit Points，HP）和耐力值（Stamina Points，SP）
耐力值表示你能承受多少伤害而不会受伤，而生命值表示你能受伤到什么程度才会昏迷或死亡。耐力值在生命值之前被消耗，而且比生命值更容易恢复。有关更详细的解释，请参见第22页。

### 等级（Level）
等级表示游戏中相对的强度。有几种类型的等级。职业等级（Class Level）是角色在特定职业中拥有的等级数。角色等级（Character Level）是角色在所有职业中拥有的等级总和。等级还可以指法术的等级、物品的等级或其他随游戏规则框架调整的机制。

:::info[译注]
在中文社区中，为区分于职业等级和角色等级，法术等级（Spell Level）全部被译作法术环级。
:::

### 调整值（Modifier）
调整值是添加到掷骰结果中的数字，例如攻击掷骰、豁免掷骰或技能检定。它可以是正数也可以是负数。

### 怪物（Monster）
怪物是一个非玩家角色。一般来说，怪物太过奇特或智力不足，无法成为玩家角色，或者因为其他原因无法成为玩家角色。怪物可能是玩家角色的对手或盟友，或者起到其他作用。

### 非玩家角色（Nonplayer Character，NPC）
非玩家角色是由游戏主持人控制的，目的是与玩家互动并推动故事发展。

### 玩家角色（Player Character，PC）
这是由玩家控制的角色。

### 角色扮演（Roleplaying）
描述角色的行为，通常是从角色的角度进行表演，这称为角色扮演。当玩家从角色的角度说话或描述行为时，就被称为“角色内”行动。

### 轮（Round）
在战术战斗中，轮是游戏世界中等于6秒的时间单位；每个能够行动的角色每轮都会有一次行动机会。在星舰战斗中，每轮由三阶段的行动组成，不对应特定的时间长度。

### 豁免掷骰（Saving Throw）
豁免掷骰是一个d20的掷骰，表示角色尝试避免或减少某种有害效果的行为。

### 层级（Tier）
层级表示规模扩展，类似于“等级（Level）”一词；它用于衡量计算机和星舰，以及其他元素。
