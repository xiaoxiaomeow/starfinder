# 这本书里有什么？

:::origin[[英文原文](https://www.aonsrd.com/Rules.aspx?ID=7)]
:::

无论你是玩家还是游戏主持人，这本书包含了游玩 Starfinder 所需的全部信息。有些人可能希望直接从第二章开始学习规则和创建角色，而另一些人可能想先了解 Starfinder 的世界设定，可以从第十二章入手。

对于需要创建角色的玩家，[第二章](/core_rulebook/chapter%202)提供了逐步创建角色的指南，并指引至相关章节。[第三章](/core_rulebook/chapter%203)介绍了可供选择的核心种族，[第四章](/core_rulebook/chapter%204)则介绍了决定角色技能和能力的职业。第[五](/core_rulebook/chapter%205)、[六](/core_rulebook/chapter%206)、[七](/core_rulebook/chapter%207)章包含进一步自定义角色能力和装备的信息，[第十章](/core_rulebook/chapter%2010)则介绍了魔法和法术，适用于具有超自然元素的角色。在开始创建角色前，可以根据需要阅读部分或全部相关章节。更多详情请参阅第 14 页的角色创建第一步。

除了角色创建相关信息，这本书还包含游玩 Starfinder 角色扮演游戏所需的规则。战术战斗、移动及相关规则是 Starfinder 的重要组成部分，星舰战斗也是其中关键内容，这些内容可在[第八章](/core_rulebook/chapter%208)和[第九章](/core_rulebook/chapter%209)找到。玩家在学习如何游玩 Starfinder 时，最好先阅读这些章节，而游戏主持人更需要熟悉它们，以确保游戏顺畅进行。

游戏主持人还应阅读[第十一章](/core_rulebook/chapter%2011)，该章节收录了 GM 相关的重要规则，例如 Starfinder 常见的环境、陷阱和毒素等危害、遭遇设计规则，以及游戏准备与运行指南等。此外，游戏主持人还需要熟悉[第十二章](/core_rulebook/chapter%2012)介绍的世界设定。如果熟悉 Pathfinder 角色扮演游戏，并希望将其中元素融入 Starfinder 冒险，还可以参考[第十三章](/core_rulebook/chapter%2013)，该章节讲解了如何将旧版内容整合进 Starfinder 规则体系。

除了本书，游玩 Starfinder 还需要一些基础物品。最重要的是，游戏主持人需提前做好准备，玩家也需要事先创建好角色（本书附录及 paizo.com 提供了[空白角色卡](https://paizo.com/download/starfinder/PZO7101-CharacterSheet.zip)）。还需要铅笔和一组多面骰，每种骰子按照“d”加上其面数命名（例如四面骰为 d4），完整的骰子组合包括 d4、d6、d8、d10、d12 和 d20，还需要一组百分骰（“d%”），可用两个十面骰模拟。

此外，还需要战术战斗地图（1 英寸方格）和星舰战斗地图（1 英寸六边形），以及用于表示角色和星舰的棋子或模型。如果不确定从哪里入手，[Starfinder Flip-Mat: Basic Terrain](https://paizo.com/products/btpy9tmh) 和 [Starfinder Flip-Mat: Basic Starfield](https://paizo.com/products/btpy9tmj) 提供了基础地图，而 [Starfinder Core Rulebook Pawn Collection](https://paizo.com/products/btpy9tml?Starfinder-Core-Rulebook-Pawn-Collection) 提供了纸质棋子，这些都可以在 [paizo.com](https://paizo.com/) 上找到，同时还有骰子套装和其他游戏配件。
