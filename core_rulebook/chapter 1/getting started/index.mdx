# 入门指南

:::origin[[英文原文](https://www.aonsrd.com/Rules.aspx?ID=1)]
:::

:::starfinder
在 Starfinder 中，你和朋友们将扮演一艘星舰的船员，探索这个奇异宇宙的种种奥秘。然而，在这个框架之下，你可以扮演任何角色，讲述无限可能的故事。你会加入寻星者协会（Starfinder Society），发掘外星科技，还是作为企业雇佣兵追逐名利？或许你是一名誓言守护新行星生态的“异种守护者”，一位能读心的神秘侦探，亦或是手持魔法剑、试图赎罪的仿生人刺客。无论你的使命为何，你和你的团队都需要运用一切魔法、武器与智慧才能生存下去。但最重要的是——你们需要彼此的支持。
:::

在你拿起电弧手枪，冲向冒险之前，有一些关于如何主持或参与 Pathfinder 游戏的关键事项你需要了解。如果你已经有角色扮演游戏的经验，可以直接跳到下一页。
