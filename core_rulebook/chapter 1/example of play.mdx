# 游戏范例

:::starfinder
GM欧文正在带领五名玩家进行他们最新的冒险。罗布扮演的是科普帕利，一名达马亚拉申塔机械师，身边带着她的宠物无人机VV-R9。朱迪扮演的是科斯切克，一名喜欢手榴弹的希伦外交官。阿曼达扮演的是佩塔尔，一名不断进行修理的尹索齐科技法师，她和詹姆斯扮演的忧郁维斯克战士奥博扎亚是最好的朋友。团队的最后一员是克里克斯，杰森扮演的人类太阳战士，一个自恋的现实广播明星。
:::

:::vanilla
这五位英雄接受了阿巴达公司（AbadarCorp）的合同，前往一颗新发现的星球探索一座古老的神庙，并带回一台神秘的机器，初步勘测显示它位于神庙的核心。他们还听到了一些传闻，可能不仅仅是他们一支队伍在寻找这个神器……
:::

**GM**：你们现在身处一条漫长而黑暗的走廊，墙壁上布满了奇怪的触手图案。在走廊的尽头有一扇门。  
**克里克斯**：我走在前面，让我的星芒武器的光芒照亮周围的区域。  
**佩塔尔**：呃，克里克斯？你知道我们中的一些人能在黑暗中看见东西吧？你的光亮基本上就是一个巨大的“来吃我”的招牌。  
**奥博扎亚**：我是最强的战士，打头阵的荣耀属于我。我走向那扇门。  
**GM**：当你靠近时，你发现门上有一种由尖刺和齿轮构成的复杂锁机械。  
**奥博扎亚**：这里没有荣耀，只有懦夫的锁。  
**科普帕利**：来，奥博，我来搞定。我研究一下这个锁，试着破解它。  
**GM**：听起来不错。进行一次工程检查。  

:::vanilla
科普帕利掷出了一个d20，结果是12。她加上了她的工程技能调整值+9，总共得到了21。GM查阅了他的笔记，发现锁的难度等级是DC 20，这意味着科普帕利的检查结果足够成功，能够打开这扇门。
:::

**GM**：你成功地找到了符文中的一个规律，并按顺序按下了其中三个符文。随着你按下它们，门滑动向后并侧面打开，通向一个新的房间。  
**奥博扎亚**：现在荣耀属于我！我走进去。

:::vanilla
GM在战斗地图上画出了一个大房间，里面有几个壁龛和一扇通向外面的门。
:::

**GM**：奥博，进入后你看到一个高达20英尺的房间，半高处有一个阴暗的阳台。房间中央有一尊雕像，看起来像是一只巨大的鸟，翅膀却被触手取代，眼睛处有三颗绿色的宝石。它旁边的地面上散落着一具穿着现代盔甲的人形尸体。你们其他人要进房间吗？

:::vanilla
玩家们商量后决定进入。GM要求他们在战斗地图上放置小型模型，以显示他们站的位置。
:::

**科斯切克**：我仔细研究一下尸体，能认出什么吗？  
**GM**：进行生命科学和文化检定。

:::vanilla
生命科学检定是为了确定尸体是什么样的生物，而文化检定则是为了看科斯切克是否能认出尸体的服装或其他文化标志。科斯切克为每个检定掷了d20，并加上相应的调整值，结果生命科学检定得到了7——不是很高——但文化检定则得到了23。GM决定她在第一次掷骰时失败了，但第二次成功。
:::

**GM**：老实说，对你来说，所有哺乳动物看起来差不多，但你能看出这具尸体是被射杀的，而且你认出了它盔甲上的标志——它是天火军团的成员。  
**科斯切克**：我告诉其他人。你们觉得这是我们的竞争对手吗？  
**佩塔尔**：切，天火军团可不会抢夺神庙的东西——那是留给我们这些小兵的。我要调查一下雕像。  
**GM**：进行一次察觉检定。

:::vanilla
佩塔尔掷出了一个d20，得到了12，然后加上她的感知调整值+3，总共得到了15。雕像上有一个魔法陷阱，佩塔尔本可以察觉到并避免它，但它的察觉DC为20，所以她没能发现并触发了陷阱。这个陷阱设计为在第一次接触时触发，从雕像的眼睛中射出闪电，给触发它的生物造成2d6点电击伤害。GM掷了2d6，结果是5点伤害，但陷阱允许进行反射豁免掷骰以减半伤害——如果佩塔尔成功，她就能躲避并减少伤害。
:::

**GM**：当你触摸雕像时，它的眼睛宝石亮了起来，房间充满了幽灵般的尖叫声。进行一次反射豁免掷骰。  
**佩塔尔**：哎呀！又来了！

:::vanilla
佩塔尔掷了一个d20，得到了6，然后加上她的反射豁免调整值+3，总共得到了9——不足以战胜陷阱的豁免DC 15，所以她受到了全额伤害。  
:::

**GM**：绿色的闪电从宝石中爆发出来，将你击飞到房间另一边，所有的毛发都竖了起来。你受到5点电击伤害。  
**科普帕利**：不错的发型，老鼠。  
**佩塔尔**：太无礼了！不过，至少宝石是真的很闪亮。  
**GM**：你们听见从对面走廊传来奔跑的脚步声。突然，六七个穿着盔甲环境套装的哥布林冲进了房间！  
**佩塔尔**：哥布林？有人派哥布林来找神器？  
**科普帕利**：嘿，阿巴达公司派了只老鼠。  
**GM**：是时候战斗了！进行先攻检定。

:::vanilla
为了确定战斗顺序，每位玩家掷一个d20并加上她的先攻调整，GM也为哥布林群体掷骰。现在他们进入战斗了，先攻检定的结果决定了各方参与者在整个战斗中的行动顺序。GM从先攻值最高的人开始，依次进行下去。奥博扎亚掷出了18，克里克斯掷出了12，哥布林掷出了10，科斯切克掷出了8，科普帕利掷出了7，而佩塔尔掷出了3。
:::

**GM**：奥博，你先动。  
**奥博扎亚**：我解开自动瞄准步枪的保险，朝最近的哥布林开火！

:::vanilla
奥博扎亚掷出了一个d20，得到了13，然后加上她使用该武器的+5攻击加值，总共得到了18。由于她的武器发射弹药并造成穿刺伤害，因此她需要对抗哥布林的动能护甲等级（KAC），它是11。因为她的攻击结果大于等于哥布林的KAC，所以她命中目标！GM告诉奥博扎亚掷伤害，于是她掷了1d6，结果是2（武器伤害），再加上她的武器专精加值+3，总共造成了5点伤害。
:::

**GM**：子弹穿透了哥布林的肩膀，但它居然还站着。  
**奥博扎亚**：哈哈！第一滴血是我的！  
**GM**：克里克斯，到你了！  
**克里克斯**：科普帕利，确保你的无人机镜头拍到我冲向前去，用我的星芒剑斩下一个哥布林的头！粉丝们一定会喜欢这个！

:::vanilla
克里克斯在战棋板上数出了自己和最近的哥布林之间的5英尺方格。他选择的最快路线是直线跨过2个方格，然后再斜着跨过2个方格。每跨过一个方格需要5英尺的移动，而斜着跨过2个方格则需要15英尺的移动，这使得他的总移动距离为25英尺——仍然低于他最大30英尺的移动范围，意味着他可以在这一回合内完全接近哥布林。他掷出了一个d20，得到了5，然后因为这是近战武器，他加上了基本攻击加值+3和力量加值+3，总共得到了11，恰好足够命中哥布林的KAC 11！他掷了武器伤害的1d6，再加上力量加值和武器专精的加值，总共造成了11点伤害。
:::
 
**GM**：你划过了哥布林的盔甲，就像划开果冻一样！它看起来很惊讶，然后倒在了地上。  
**克里克斯**：我转身面向镜头，摆出姿势，拿着发光的金色刀刃，然后低声说我的标语：“金光闪闪！”  
**GM**：别太得意了，因为现在轮到哥布林了。而且，由于你刚刚表现得威胁十足，最大的那个哥布林正朝你冲过来！它不知怎么拿到了一把超大激光步枪，正在瞄准你。

:::vanilla
GM为第一个哥布林投掷攻击，使用克里克斯的能量护甲等级（EAC），因为这是激光攻击，结果是自然20——一次重击！GM投掷正常伤害骰子两次——2d8而不是1d8，结果是13——并添加了1d6燃烧伤害（投掷结果为2）。克里克斯现在受到13点伤害，然后每回合都受到2点燃烧伤害，直到火焰被扑灭。
:::

**GM**：克里克斯，那激光击中了你完美的下巴。你受到15点伤害，你盔甲上的装饰性花纹熔化并着火——你将在接下来的每回合受到2点伤害，直到扑灭火焰。你还活着吗？

:::vanilla
克里克斯查看了他的角色表。他有11点耐力值和9点生命值。他首先从耐力值中扣除伤害，耗尽了所有11点，然后从生命值中扣除剩余的4点伤害，剩下0点耐力值和5点生命值。
:::

**克里克斯**：呃！真是侥幸。我大喊：“我的脸！我的美丽脸！”  
**科斯切克**：撑住，克里克斯——你的粉丝需要你！  
**克里克斯**：科斯切克，如果我挺不住了……一定要确保我的头发好看……
