# 什么是角色扮演游戏？

Starfinder是一款桌面冒险角色扮演游戏（RPG）：这是一种互动故事，其中一名玩家——游戏主持人（Game Master）负责设定场景并呈现挑战，而其他玩家则扮演科幻幻想英雄，努力克服这些挑战。通过根据角色的个性和能力应对不同情况，玩家们帮助构建故事情节，因为每个场景（称为“遭遇”）的结果将推动故事进入下一个阶段。

掷骰子和预设的统计数据结合在一起，增添了随机性，并决定角色是否能够成功完成他们的行动。你可以将角色扮演游戏想象为一场戏剧：玩家是演员，而游戏主持人是导演。但你并不需要成为一个熟练的演员或讲故事的人才能玩这个游戏；只需要描述你希望角色做的事情，让游戏主持人和规则来完成其余的部分！
