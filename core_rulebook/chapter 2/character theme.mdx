# 角色主题

:::starfinder
角色主题代表了你的角色的一个核心焦点，这可能是你的背景、成长经历、训练或神秘命运的结果。你在1级时选择一个角色主题，并在1级、6级、12级和18级时获得反映该主题主要方面的特殊奖励。除非另有说明，所有能力均为特异能力。在GM的允许下，你甚至可以创建完全符合你角色概念的自定义主题！
:::

:::info[编注]
每个主题的细节被统一整理，参见[主题](/themes)。
:::

:::vanilla[[王牌飞行员](/themes/ace%20pilot)]
凭借稳定的双手和钢铁般的神经，你擅长操作星舰和其他载具。
:::

:::vanilla[[赏金猎人](/themes/bounty%20hunter)]
几乎没有什么能阻止你追踪目标并将其带回，无论是死是活。
:::

:::vanilla[[偶像](/themes/icon)]
在殖民星域的范围内，你是一位受欢迎且受人尊敬的明星。
:::

:::vanilla[[雇佣兵](/themes/mercenary)]
你是一位训练有素的雇佣兵，在战斗中与同伴配合默契。
:::

:::vanilla[[法外狂徒](/themes/outlaw)]
无论你是否真的有罪，你在某个城市、星球甚至整个星系中都是一个通缉犯。
:::

:::vanilla[[祭司](/themes/priest)]
你对某种哲学或宗教的坚定信仰构成了你个性的核心。
:::

:::vanilla[[学者](/themes/scholar)]
作为一名学者，你拥有广泛的知识基础，并渴望不断扩展它。
:::

:::vanilla[[星际旅者](/themes/spacefarer)]
你在星际间生活，寻找新的世界去探索，并渴望下一次冒险。
:::

:::vanilla[[异形探索者](/themes/xenoseeker)]
当你离开契约世界星域时，你努力与外星生命形式建立联系。
:::

:::vanilla[[无主题](/themes/themeless)]
你不完全符合上述任何类别，或者你将自己视为一张白纸。
:::
