# 属性值

:::origin[[英文原文](https://www.aonsrd.com/Rules.aspx?ID=41)]
:::

你的角色有六项属性值，这些属性值代表她的基本能力和原始潜力：力量、敏捷、体质、智力、感知和魅力。这些属性值几乎会影响你角色的所有能力——例如，敏捷决定她的灵活性和瞄准的稳定性，智力代表她的教育水平和推理能力，等等。（关于每项属性的具体描述，请参见[属性描述](#属性描述)部分。）属性值通常在3到18之间，平均值为10。

不同的属性值对你的重要性取决于你想扮演的角色类型，而每个职业都有一个关键属性值，该属性值是该职业特性的核心，帮助你获得完成该职业职能的能力。第19页的表2-3列出了每个职业的关键属性值；每个职业描述的第一页也会列出该职业的关键属性值，以及其他对该职业有帮助的属性值，尽管这些属性值可能没有那么重要。

以下是 Starfinder 默认的属性值生成方法，称为购点系统。这些规则允许你自定义属性值，以构建你理想中的角色。此外，我们还提供了两种可选系统：快速序列系统让你从几个预设的属性值序列中选择，以便快速便捷地创建角色；而高级玩家可能会对骰点系统感兴趣，这种方法通过随机生成属性值来增添乐趣，有时会带来显著的强项和弱项。

你只需要使用其中一种系统——请与你的GM确认她是否有偏好的系统。如果有疑问，使用购点系统。

## 购点系统

在这种方法中，你可以通过使用点数池“购买”属性值来自定义你的属性值。由于该系统的目的是帮助你构建你理想中的角色，因此在开始自定义属性值之前，请先确定你角色的种族（[第三章](/core_rulebook/chapter%203)）、职业（[第四章](/core_rulebook/chapter%204)）和主题（见第28页）。一旦你明确了这些，请按以下步骤操作。

**第1步：每个属性值从10开始。** 在你的角色表或草稿纸上，写下所有六项属性值——力量、敏捷、体质、智力、感知和魅力——并在每项旁边写上10。

**第2步：根据种族增减点数。** Starfinder 中的种族通常在某些属性上具有天赋，而在其他属性上则稍显不足。每个种族描述的第一页会列出这些优势与劣势，具体表现为对特定初始属性值的增减点数；你也可以查看表2-2，快速了解所有种族的优势和劣势。例如，一个希伦种族开始时会在体质和感知上获得+2点，但在魅力上会获得-2点（这意味着你需要将初始体质和感知值加2点，但将初始魅力值减2点）。某些种族，例如人类，非常全能，因此可以将点数分配到任意一项属性值上。

一旦你知道了你的种族调整值，就从初始的10点中加上或减去这些点数。  

**第3步：根据主题增加点数。** 每个主题会为你提供1点属性值，可以加到某一项属性值上。例如，选择王牌飞行员主题会为你的敏捷+1点，而无主题选项则允许你将这1点加到任意一项属性值上。每个主题提供的点数可以在其描述或表2-2中找到。一旦你确定了主题，就将该点数加到指定的属性值上。

**第4步：花费10点自定义你的属性值。** 现在你已经有了包含种族和主题调整后的基础属性值，接下来你会获得10点额外的点数池，可以按照你的意愿分配到属性值上。这些点数以1:1的比例加到现有的属性值上——例如，如果你的敏捷值为12，而你从点数池中加了1点，那么你的敏捷值将变为13。你可以随意分配这些点数，但不能让任何单项属性值超过18。（之后，随着角色升级和获得提升属性值的装备，你的属性值可能会超过18，但18是任何角色初始属性值的上限。）请确保用完所有的10点属性值——你不能将它们留到以后使用。

**第5步：记录属性值及属性调整值。** 一旦你用完所有点数，就完成了。将最终的属性值填写到角色表的相应位置，然后查看表2-1，找到每项属性值对应的调整值，并将其记录下来。调整值的解释请参见第21页的“属性调整值与属性检定”。

### 购点示例

假设你正在构建一个维斯克士兵——你看到了第27页的插图，立刻开始想象一个粗犷但忠诚的雇佣兵，她按照自己的规则行事。

像所有角色一样，你从每项属性值10开始。你已经决定选择维斯克种族，查看表2-2后，你发现Vesk种族在力量上获得+2点，体质上获得+2点，但在智力上获得-2点。将这些调整值应用到初始的10分上，现在你的属性值如下：

:::vanilla
力量12，敏捷10，体质12，智力8，感知10，魅力10
:::

接下来要应用一个主题。你希望她的主题是雇佣兵，从表中可以看到，雇佣兵主题会为力量+1点。调整后的属性值如下： 

**第3步：根据主题调整属性值**  

:::vanilla
力量13，敏捷10，体质12，智力8，感知10，魅力10
:::

到目前为止，你的角色完全符合你的设想——非常强壮但不太聪明。现在是时候拿出10个自由点数来分配乐。表2-3告诉你，士兵的关键属性值是力量（用于近战）或敏捷（用于投射武器）。你想象她挥舞着一把巨大的突击锤冲入战斗，因此你决定花费5点将她的力量从13提升到18（这是她这个等级能达到的最高值）。同时，你知道在某些情况下她会需要使用枪械，因此你又花费4点将她的敏捷提升到14。现在还剩下1点，你考虑将其加到智力上以抵消种族劣势，但最终决定将其加到魅力上——她可能是个粗人，但她应该有自己的魅力。最终的属性值如下：

:::vanilla
力量18，敏捷14，体质12，智力8，感知10，魅力11
:::

你将最终的属性值填写到角色卡上，然后根据表2-1找到每项属性值对应的调整值并记录下来——18等同于+4调整值，14等同于+2，等等。现在，你已经准备好继续完成角色的其他部分了！
