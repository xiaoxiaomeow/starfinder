# 按步骤创建角色

:::origin[[英文原文](https://www.aonsrd.com/Rules.aspx?ID=31)]
:::

:::starfinder
作为 Starfinder 角色扮演游戏的玩家，首先你需要创建一个角色。这是你在游戏中扮演的虚拟人物，你需要为她设计背景和个性。你还需要通过游戏的机制来构建角色，确定她在各种任务中的初始能力，以及她在冒险中可以使用的特殊技能。
:::

:::pencil[角色卡]
在下面详细描述的每个角色创建步骤中，填写角色卡上相应字段中的必要信息。你可以在本书的末尾找到[空白角色卡](https://paizo.com/download/starfinder/PZO7101-CharacterSheet.zip)，或者在 [paizo.com](https://paizo.com/) 网站上免费下载。
:::

以下说明假设你正在创建一个1级角色。这些步骤按建议的顺序排列，但你可以按自己的顺序完成它们。

许多下面的步骤会指示你在角色表格上填写相应字段，但请注意，角色表格的设计是为了方便游戏过程中的使用，而不是角色创建。例如，角色表格顶部列出的一些字段是你在角色创建过程结束时才确定的详细信息。

如果你正在创建一个更高等级的角色，建议你先按照这些说明进行操作，然后再转到第26页的升级角色的说明。

## 第1步：创建角色概念

你想扮演什么样的科幻英雄？角色创建的一个好方法是决定你角色个性的大致方向，勾画出她的一些背景细节，并思考她为什么以及如何进行冒险。在这个步骤中，你可能会想浏览 Starfinder 中可用的种族、主题和职业，以便了解可选项。下方的侧边栏简要介绍了这些内容，它是一个很好的起点；有关种族的完整信息在[第三章](/core_rulebook/chapter%203)和[第十三章](/core_rulebook/chapter%2013)，职业在[第四章](/core_rulebook/chapter%204)介绍，第28–37页详细说明了主题。种族指的是你的物种——从像昆虫一样的希伦和爬行动物一样的维斯克，到人类和精灵。主题代表你背景中的一个元素，无论是来自你的成长、训练还是命运；它可以与职业相关，但不一定需要。职业代表角色在战斗和冒险中的主要能力方向——例如，技术法师操控科技并施法，外交官发布命令并激励盟友进行英雄壮举，等等。

一旦你考虑了选项，简要描述一下你的角色，重点考虑她擅长什么（射击枪支、驾驶星际飞船等）。然后思考如何根据本书的规则组件最好地模拟这一点。如果你想玩一个手持巨型激光步枪的身形庞大的爬行动物角色，那么选择一个维斯克士兵配上雇佣兵主题是一个完美的选择。如果你更喜欢一个灵活的双足老鼠，喜欢盗窃，那么选择一个尹索齐特工并搭配法外狂徒主题可能是个不错的选择。如果你感到迷茫，可以试着从书中的某个艺术插图中选择一个角色，并创建一个与之匹配的角色。

:::pencil[角色卡]
在“玩家”字段中写下你的名字。根据你对角色背景的开发程度，你还可能想记下角色的名字和性别，甚至可以添加一段简短的外貌描述或草图。
:::

## 第2步：选择种族

你角色的种族或物种是她最重要的特征之一。[第三章](/core_rulebook/chapter%203)和下方的边栏详细介绍了 Starfinder 的核心种族——其中许多种族非常异乎寻常——而[第十三章](/core_rulebook/chapter%2013)则介绍了与剑与魔法角色扮演游戏相关的经典种族。你角色的种族赋予她特定的能力和语言，并决定了她在开始游戏时拥有的部分生命值（HP，表示她在昏迷或死亡前能够承受的伤害）。她的职业也会影响她的初始生命值总量，如[第6步](#第6步应用职业)所述。

你角色的种族还会影响她的属性值。这些是影响你角色多个数值的重要数值，但你将不会在[第5步](#第5步确定属性值)之前最终确定它们。

:::starfinder[种族]
**安卓人（Android）**：拥有机械组件的人造生命体，曾经被打造为仆人，但现在被视为公民。  
**人类（Human）**：极其多才多艺、适应力强的种族，始终在扩张和探索。  
**卡萨达（Kasatha）**：来自遥远沙漠世界的四臂种族，拥有高度传统的文化。  
**拉申塔（Lashunta）**：具有魅力和心灵感应能力的学者种族，分为两种亚种：一种高大纤细，另一种矮小结实。  
**希伦（Shirren）**：昆虫状的种族，曾从类似蝗虫的蜂巢中脱离，强调集体主义，但又沉迷于个人选择。  
**维斯克（Vesk）**：好战的爬行动物种族，最近与其他种族宣告停战——暂时如此。  
**尹索齐（Ysoki）**：也称为“鼠族”，这些矮小、毛茸茸的拾荒者以其大个性弥补了身高的不足。
:::

:::pencil[角色卡]
在角色表格的顶部填写角色的种族。记录她的初始种族生命值和种族语言（如果有的话），以及她可以使用的种族能力。此时，请在属性值旁边的空白处记录她因种族而增加或减少的属性值点数（你将在[第5步](#第5步确定属性值)中用到这些！）。
:::

## 第3步：选择主题

接下来，你可以为角色选择一个主题，主题代表她背景和动机的核心方面。主题可以帮助表达角色冒险的方式——无论她是赏金猎人、学者，还是另一个经典冒险角色的体现。除了反映某种背景外，主题还会在1级时提供对某个技能或多个技能的加成，并且会为某个特定的属性值增加1点——你将在[第5步](#第5步确定属性值)中最终确定你的属性值。

Starfinder 的角色主题在下方的边栏中有简要总结，详细信息从第28页开始，此外也包括了无主题的规则。

:::starfinder[主题]
**王牌飞行员（Ace Pilot）**：擅长操作星际飞船和其他交通工具的人，痴迷于所有相关的知识和传说。  
**赏金猎人（Bounty Hunter）**：无可阻挡的追踪者，擅长追捕逃亡者。  
**偶像（Icon）**：受人喜爱和尊敬的名人，能够利用公众的崇拜为自己谋取利益。  
**雇佣兵（Mercenary）**：训练有素的财富兵，既能作为战斗员也能作为小队指挥官。  
**法外狂徒（Outlaw）**：被通缉的罪犯，拥有黑市和不法交易的后路，能够应对法律麻烦。  
**祭司（Priest）**：虔诚且博学的信徒，能够在其他追随者中建立威信。  
**学者（Scholar）**：擅长研究和前沿思维的知识分子，拥有广泛的知识和渴望扩展它的愿望。  
**星际旅者（Spacefarer）**：不安分的探索者，直觉敏锐，收集了大量关于外星生物学和拓扑学的知识。  
**异形探索者（Xenoseeker）**：外星生命体的专家，认为遇见它们是人生最有意义的成就之一。  
**无主题（Themeless）**：不属于上述任何类别的角色，开辟出一条个人的决心和训练之路。
:::

:::pencil[角色卡]
在角色表格的顶部写下角色的主题，并记录它为技能提供的加成。此时，请在属性值旁边的空白处记录它为某个属性值增加的点数（你将在[第5步](#第5步确定属性值)中用到这些！）。
:::

## 第4步：选择职业

在这个阶段，你已经快要最终确定角色的属性值了，这些属性值是计算许多角色属性的关键数值。不过，在此之前，你需要先选择角色的职业。职业决定了她能使用哪些英雄能力，影响她的战斗能力，并决定她抵抗或躲避某些有害效果的难易程度。[第四章](/core_rulebook/chapter%204)详细介绍了 Starfinder 中的七种可选职业，下方的边栏则提供了各职业的功能和定位概述。

目前，你不需要详细记录角色的所有职业特性。你只需要确定自己要玩的职业，它将告诉你哪些属性值对你来说最重要。每个职业都会列出其主要属性值以及其他特别有用的属性值。为了方便查阅，表2-3（第 19 页）列出了所有职业的关键属性值。

:::starfinder[职业]
**外交官**：具有魅力的社交达人，擅长多种技能，能够激励盟友完成伟大的英雄壮举。  
**机械师**：机器和技术的专家，通过修理制造出无人机伙伴或强大的脑部植入物。  
**秘术师**：魔法使用者，与强大力量之间的神秘联系赋予她打破宇宙法则的能力。  
**特工**：隐秘的战士，拥有广泛的知识，擅长利用敌人的疏忽。  
**太阳战士**：纪律严明的战士，掌握星际力量，能够获得由星辰能量制成的武器或盔甲。  
**士兵**：精通各种盔甲、枪械和近战武器的专家，专注于某些类型的装备。  
**科技法师**：魔法使用者，具有与技术的超常共鸣能力，能利用技术解锁强大的效果。
:::

:::pencil[角色卡]
在你的角色卡的顶部写下你的角色职业，并在后面标注“1”，表示你是1级角色。不用担心职业带来的其他能力——这些内容会在[第6步](#第6步应用职业)处理。
:::

## 第5步：确定属性值

现在，你已经做出了关于角色的一些重要决定，是时候填写她的各项属性了。角色的属性值决定了她在游戏中的多种能力，并由以下六项组成：力量（Strength）、敏捷（Dexterity）、体质（Constitution）、智力（Intelligence）、感知（Wisdom）和魅力（Charisma）。

要了解如何计算你的属性值，请参阅第18页的“属性值”部分。该部分还会指导你如何确定属性值调整值，这些调整值会影响角色卡上的许多数值，并在本节后续内容中被使用。

:::pencil[角色卡]
在角色卡的相应栏位填写你的属性值，并在每个属性值旁边记录对应的调整值。
:::

## 第6步：应用职业

现在，你需要确定你的角色在1级时，从所选职业获得的力量、特殊能力（称为职业特性） 以及其他关键数值。

每个职业章节开头的表格列出了角色的基础攻击加值，它决定了角色有多擅长攻击。你需要将这个基础数值添加到近战和远程攻击掷骰中，关于战斗的更多详情请参阅[第八章](/core_rulebook/chapter%208)。该表格还列出了角色的强韧（Fortitude）、反射（Reflex）和意志（Will） 豁免的基础加值，详细信息请参考[第9步](#豁免掷骰)。

此外，角色的职业会影响她的耐力值（SP）总量。耐力值表示角色在受到真正伤害之前可以承受多少攻击。在1级时，你的耐力值等于职业表格中列出的基础耐力值+体质调整值。在1级时，还会把职业表格中的基础生命值加上你的种族提供的生命值（关于生命值与耐力值的更多信息，请参见第22页）。

最后，角色的职业会影响她每个等级可分配的技能点（见[第7步](#第7步分配技能点并选择专长)），以及她的武器与防具熟练度（[第8步](#第8步购买装备)）。

:::pencil[角色卡]
记录角色的总耐力值总生命值。记录角色的基础攻击加值以及职业提供的基础豁免加值。（角色最终的豁免调整值计算方式将在[第9步](#豁免掷骰)讲解。）记录1级时获得的职业特性。
:::

## 第7步：分配技能点并选择专长

接下来，你需要决定角色擅长哪些技能，例如使用科技还是在荒野中生存。每个等级，角色都会获得技能点（Skill Ranks），代表她不断增长的知识和训练。每个等级获得的技能点等于智力调整值+职业提供的技能点数（但至少获得1点）。你可以自由分配这些点数到任何技能上。部分技能被标记为职业技能（Class Skills），这些是与你的职业紧密相关的技能。如果你在这些技能上投入至少1点技能点，你会自动获得 +3 加值。你仍然可以投入点数到非职业技能，只是这样做不会获得额外的 +3 加值。关于技能的详细信息，包括仅限受训技能和穿戴防具对技能的惩罚，请参阅[第五章](/core_rulebook/chapter%205)。

在分配技能点后，选择角色的专长（Feats）。大多数角色在游戏开始时获得1项专长，而人类种族的角色会获得额外的种族奖励专长。专长可以很好地强化角色的可能缺少的能力。例如，坚韧（Toughness）可以增加额外的耐力值，保镖（Bodyguard）能让你能够保护附近的盟友免受攻击。关于专长的更多信息，请参阅[第六章](/core_rulebook/chapter%206)。

:::pencil[角色卡]
在角色卡上分配技能点，注意单个技能的点数永远不能超过角色等级！如果这些中某些技能是职业技能，请在相应栏位添加 +3 职业技能加值。在每个技能中加入属性调整值。记录角色的专长，如果某些专长提供技能加值，请在技能栏的其它调整值里注明。最终，计算并记录所有非“仅限受训”的技能的总调整值。
:::

## 第8步：购买装备

在1级时，你的角色有1000点信用点可以用来购买防具、武器和其他补给（尽管你的GM可能会给你其他指示来选择装备）。护甲和武器通常是最重要的——请记住，你的角色职业决定了她擅长的武器和护甲类型。角色的武器决定了她在战斗中能造成的伤害，而防具则影响她的能量护甲等级（EAC）和动能护甲等级（KAC）——更多关于护甲等级的信息，请参见[第9步](#护甲等级)。此外，还有一些其他常用物品可以在游戏过程中帮助你的角色。有关可用装备及其价格的更多信息，请参阅[第七章](/core_rulebook/chapter%207)。

:::pencil[角色卡]
当你花费角色的信用点后，请记录她所拥有的装备以及剩余的信用点。在相应的行上记录你的武器及其相关细节，包括每件武器造成的伤害（有关如何计算伤害的详细信息，请参见第240页）。
:::

## 第9步：填写最终细节

现在将以下细节添加到你的角色卡中。

### 阵营
记录你的角色的阵营，这会影响她的个性。更多信息请参见第24页。

### 护甲等级
能量护甲等级（EAC）和动能护甲等级（KAC）代表你的角色在战斗中被击中的难度。能量护甲等级代表她在受到能量攻击（如激光）时避免伤害的能力。动能护甲等级代表她在受到物理攻击（如子弹）时避免伤害的能力。（更多信息请参见第240页。）

EAC和KAC的初始值均为10。将你的角色的敏捷调整值分别加到这两个值上，然后再加上她的护甲提供的EAC加值和KAC加值。你还需要记录你的角色对抗战技的护甲等级，即她的KAC+8。

### 攻击加值
你在[第6步](#第6步应用职业)中已经记录了角色的基础攻击加值，但你还需要计算她在近战、远程和投掷武器上的总攻击加值。将她的力量调整值加到近战和投掷攻击上，将她的敏捷调整值加到其他远程攻击上。然后将来自专长或职业特性的其他调整值加到相应的攻击上。

### 负重能力
这是你的角色基于她的力量值可以携带的负重。当携带的负重等于或超过她力量值的一半时，她会处于中载状态（见第275页）。当携带的负重等于或超过她的总力量值时，她会处于重载状态（见第276页）。

### 信仰
记录你的角色所信仰的神祇（如果有）。更多关于 Starfinder 中宗教和哲学的信息，请参见第482页。

### 描述
如果你还没有写，请为你的角色写一段简短的描述。

### 家园
这是你的角色成长的地方，可能是一个星球、空间站，甚至是一颗小行星。（更多灵感请参见[第十二章](/core_rulebook/chapter%2012)。）

### 先攻
你的角色的先攻调整值会加到她的先攻检定中，以决定她在战斗中的行动顺序。它等于她的敏捷调整值加上来自专长或其他影响先攻的能力的调整值。

### 语言
你的角色在游戏开始时会说和读通用语、她的种族语言（如果有）以及她的家园语言（如果有）。如果她的智力调整值为正，她还会额外掌握等于该调整值数量的语言。更多关于语言的信息以及可选语言列表，请参见第40-41页。

### 决心点
你的角色可以使用决心点来完成许多重要的事情，包括恢复她的耐力值以及激活某些职业特性或其他能力。在1级时，你的角色拥有的决心点数量等于1+她的职业关键属性调整值。更多关于决心点的信息，请参见第22页。

### 豁免掷骰
有时，你的角色需要判断她是否能避免或摆脱某种效果或法术。当这种情况发生时，GM会根据情况要求进行一次强韧、反射或意志豁免掷骰。  

豁免掷骰是一个d20掷骰，你需要加上你的角色相应类型的基础豁免加值和其他适用的调整值。你的职业决定了她在每种豁免掷骰上的基础加值。此外，将体质调整值加到强韧豁免上，敏捷调整值加到反射豁免上，感知调整值加到意志豁免上——即使相关属性调整值为负也必须加上。最后，根据情况加上其他调整值，例如来自专长或其他能力的加值。

### 体型
你的角色的种族决定了她的体型（见第255页）。  

### 速度
你的角色在游戏开始时的陆地速度为30英尺，除非她有调整速度的能力（或受到防具或负重的惩罚）。更多关于速度和移动的信息，请参见[第八章](/core_rulebook/chapter%208)。

### 星舰
虽然你可以在不使用星舰规则的情况下玩 Starfinder，但大多数冒险都假设玩家角色是（或即将成为）一艘共享的星舰的船员。询问你的GM你们是否会开始就有一艘星舰，或者是否会在冒险中获得一艘星舰。如果你开始时拥有一艘星舰，请向GM索取其数值，并了解你是否可以对其进行定制——至少，每艘船都需要一个好名字！更多关于星舰的信息，请参见[第九章](/core_rulebook/chapter%209)。

一旦你将所有这些信息记录到你的角色表中，你就可以开始冒险了！
