# 第三章 种族

:::starfinder
本章概述了 Starfinder 中的“核心种族”——这些种族在契约世界中最为常见，并设计为玩家角色（PC）使用。关于一些不太常见的种族，请参阅第13章。
:::

Starfinder 角色扮演游戏不仅仅是关于遇见外星人——它还涉及扮演外星角色。在 Starfinder 中，“种族”一词通常指的是具有智慧、自我意识的物种，其成员可以被视为角色而非简单的怪物。虽然并非所有种族都适合作为玩家角色，但许多种族是可以的；任何具有种族特性条目的生物都属于潜在的可玩种族，前提是你的GM批准。

以下部分介绍了 Starfinder 中的核心种族——这些种族在契约世界中非常常见，以至于在整个太阳系中无处不在（或至少被广泛认知）。一些在契约世界系统中稍不常见的种族，如精灵和矮人，可以从第506页开始找到，而更多潜在的种族可以在 Starfinder 异族档案（Starfinder Alien Archive）中找到。请记住，这些只是契约世界中最常见的种族——系统中还有许多文明但不太突出的种族，而在契约世界之外的种族，则一切皆有可能！

选择你的种族是角色创建中最重要的选择之一，因为一旦选定，就无法更改。除了文化背景外，每个种族还附带一定数量的生命值（在1级时获得），以及其他几个种族特性，这些特性会修改你的数据或赋予你额外的能力。其中最重要的是属性调整值。这些加值和减值在生成你的属性值时适用（见[属性值](/core_rulebook/chapter%202/ability%20scores)），反映了你种族的自然天赋和劣势，例如维斯克平均比其他种族更强壮，而尹索齐则较弱。如果你已经知道自己想玩的职业，通常在选择种族时将其关键属性值（见第19页）与各种族提供的属性调整值进行比较是个好主意，这样你就不会意外地发现种族调整值让你难以玩你想玩的角色类型。某些种族的属性加值使它们非常适合某些职业，例如维斯克的力量和体质加值使它们成为天生的士兵。然而，如果一个想法让你兴奋，不要害怕打破常规——这里介绍的每个种族在其社会中都有各种职业的成员。

## 语言

契约世界的众多民族使用各种各样的语言，从被称为“通用语”的全系统贸易语言到晦涩的外星方言和来自其他现实位面的古老语言。许多世界都有共享的行星语言，大多数种族都有自己的种族语言，而契约世界中所有流行的语言都有手语版本和书面版本（包括视觉和触觉书写）。

角色开始时会说和读通用语、她的种族语言（如果有的话）以及她的家园语言（如果有的话）。她还可以根据她的智力加值从以下列表中选择相应数量的额外语言。角色可以通过在文化技能中投入技能点（见第139页）来学习她已知语言的手语或盲文版本。一个开始时失明的角色自动知道她已知语言的盲文版本；一个开始时失聪的角色自动知道她已知语言的手语版本。契约世界中使用的语言数量惊人，并非所有语言都能被其他种族理解或复制，除非借助复杂的技术；以下列出了一些契约世界中最常用的语言。

### 常见语言

通用语（Common）是契约世界中最常见的贸易语言，据信它基于葛拉利昂（Golarion）上的一种或多种古老的人类语言。契约世界中其他广泛使用的语言（及其典型使用者）包括以下这些。

- **阿基顿语（Akitonian）**：阿基顿（Akiton）的居民
- **邪灵语（Aklo）**：奥克顿（Aucturn）和黑色领域（Dominion of the Black）的居民
- **布莱塞达语（Brethedan）**：布莱塞达（Bretheda）、莱尔瓦拉（Liavara）及其卫星的居民
- **卡斯卓韦安语（Castrovelian）**，也称为拉申塔语（Lashunta）：拉申塔（lashuntas）和卡斯卓韦（Castrovel）的居民
- **埃奥克西安语（Eoxian）**：埃奥克斯（Eox）的居民
- **卡萨达语（Kasatha）**：卡萨达（kasathas）
- **希伦语（Shirren）**：希伦（shirrens）
- **特里亚克萨斯语（Triaxian）**：特里亚克萨斯（Triaxus）的居民
- **沃塞斯语（Vercite）**：沃塞斯（Verces）的居民
- **维斯克语（Vesk）**：维斯克（vesk）和维斯卡瑞姆（Veskarium）的居民
- **尹索齐语（Ysoki）**：尹索齐（ysoki）

### 其他语言

以下语言较为少见，但学者、施法者和在说这些语言的生物家园里做生意的人经常会遇到这些语言。

- **深渊语（Abyssal）**：恶魔、混乱邪恶的异界生物和深渊（Abyss）的居民
- **水族语（Aquan）**：水元素位面的居民
- **阿卡宁语（Arkanen）**：阿卡宁（Arkanen）和奥索罗（Osoro）的居民
- **风族语（Auran）**：风元素位面的居民
- **阿兹兰特语（Azlanti）**：阿兹兰特人（Azlanti）和阿兹兰特星帝国（Azlanti Star Empire）的居民
- **天界语（Celestial）**：天使、善良异界生物和善良阵营位面的居民
- **龙语（Draconic）**：龙、爬虫类人生物和特里亚克萨斯龙裔（Triaxian dragonkin）
- **卓尔语（Drow）**：卓尔和阿珀斯塔（Apostae）的许多居民
- **矮人语（Dwarven）**：矮人
- **精灵语（Elven）**：卓尔、精灵和半精灵
- **侏儒语（Gnome）**：侏儒
- **地精语（Goblin）**：熊地精、地精和大地精
- **半身人语（Halfling）**：半身人
- **火族语（Ignan）**：火元素位面的居民
- **炼狱语（Infernal）**：魔鬼、守序邪恶的异界生物和地狱（Hell）的居民
- **卡洛语（Kalo）**：卡洛人（kalo）和卡洛-马霍伊（Kalo-Mahoi）的居民
- **恩查基语（Nchaki）**：恩查克（Nchak）的居民
- **兽人语（Orc）**：兽人和半兽人
- **萨塞西安语（Sarcesian）**：萨塞西安人（sarcesians）
- **肖布哈德语（Shobhad）**：肖布哈德人（shobhads）
- **土族语（Terran）**：土元素位面的居民

## 基础生理数据

表3-1提供了一些基本范围，帮助你确定角色的身高、体重和年龄。虽然大多数角色会落在其种族范围的中间值附近，但一些特殊的个体可能会更大或更小。性别在某些种族的体型和形态中起着重要作用，但即使对于这些种族，你也应该自由地构建一个让你感觉合适的角色。环境因素也可能影响角色的体型和形态——来自低重力环境的角色可能比平均水平更高更瘦，而在高重力世界长大的角色可能由于身体承受的压力而更矮更健壮。

:::table[表3-1 不同种族的基础生理数据]
| 种族 | 平均身高 | 平均体重 | 成年年龄 | 最大年龄 |
| :----: | :----: | :----: | :----: | :----: |
| 安卓人 | 5-7尺 | 100-200磅 | 0岁 | 见第43页 |
| 人类 | 5-7尺 | 100-300磅 | 18岁 | 80+2d20岁 |
| 卡萨达 | 6-7尺 | 120-200磅 | 25岁 | 100+2d20岁 |
| 拉申塔 | 5-7尺 | 140-180磅 | 20岁 | 80+2d20岁 |
| 希伦 | 5-6尺 | 100-150磅 | 5岁 | 50+1d20岁 |
| 维斯克 | 6-8尺 | 200-300磅 | 16岁 | 70+1d20岁 |
| 尹索奇 | 3-4尺 | 60-100磅 | 10岁 | 60+1d20岁 |

表中列出的成年年龄代表了一个种族成员可能被视为成年人的年龄。这是基于生理和文化因素的概括——个别文化可能有所不同。最大年龄包括随机性以反映死亡的不可预测性，并且这是在没有魔法或技术干预的情况下对种族寿命的假设——通过适当的延寿技术，所有种族的个体都可以变得几乎永生。此外，这里介绍的大多数种族都是中型生物（见第255页）；它们的占据空间和触及范围为5尺，陆地速度为每轮30尺。虽然尹索奇是小型生物（见第255页），但他们的占据空间、触及范围和陆地速度与中型生物相同。

## 阅读种族条目

以下页面解释了核心种族的规则，并描述了它们如何融入 Starfinder 的背景设定，但有几个关键元素值得进一步解释。

- **属性调整值**：这些是基于种族的属性值调整，在角色创建时应用（见[属性值](/core_rulebook/chapter%202/ability%20scores)）。例如，安卓人的属性调整值为+2敏捷和+2智力，但-2魅力。
- **生命值**：这是你在1级时从种族获得的额外生命值。更多信息见[健康与决心](/core_rulebook/chapter%202/health%20and%20resolve)。
- **种族特性**：每个种族的第一页列出了你扮演该种族角色时获得的特殊能力。你会自动获得所有这些能力——无需选择。
- **扮演该种族**：这些注释为你提供了一个起点，帮助你了解作为该种族角色如何与世界互动。请注意，与种族条目中呈现的其他文化细节一样，这些只是基于典型种族成员的建议。个性因人而异，你的角色可能会与这些建议大相径庭，尤其是如果她在不同的文化中长大或在特殊环境下成长。

:::info[编注]
每个种族的细节被统一整理，参见[种族](/themes/themes)。
:::

:::vanilla[安卓人]
安卓人是具有生物和机械元素的人工生命体，最初由人类创造为仆人，现在则自由地在星际间谱写自己的命运。
:::

:::vanilla[人类]
人类几乎遍布 Pact Worlds 的每个角落，自从他们的母星 Golarion 消失后，他们已扩散到更远的地方。他们以好奇心、坚韧和适应能力著称。
:::

:::vanilla[卡萨达]
卡萨达是来自遥远星系的古老四臂种族，他们是坚定的传统主义者，其习俗使他们显得智慧而神秘。
:::

:::vanilla[拉申塔]
拉申塔是天生具有灵能天赋的种族，分为两个亚种：一种高挑纤细，另一种矮小强壮。两者都对其他种族具有吸引力，并致力于学术和自我完善。
:::

:::vanilla[希伦]
希伦曾是恐怖蜂巢思维的一部分，吞噬一切，但后来突变并脱离出来，成为独立但注重社区的个体，沉迷于选择的自由。
:::

:::vanilla[维斯克]
维斯克是致力于征服和统治的爬虫类种族，他们最近才结束了与契约世界其他种族的长期战争，尽管他们具有荣誉感和在战斗中的实用性，但许多人仍然不信任他们。
:::

:::vanilla[尹索奇]
尹索奇是充满激情和韧性的鼠类种族，擅长惹麻烦和摆脱麻烦。他们对技术、探索和冒险的热爱驱使他们穿越银河。
:::
